SafeZone is an early-warning and action assistant for recognizing and responding to potentially dangerous interpersonal situations. Users describe what is happening, and SafeZone provides a transparent risk level and actionable guidance.

## Key Features
- Rule-based classifier (keywords with weights) for LOW, MEDIUM, HIGH, CRITICAL.
- Tailored safety advice and suggested actions for each level.
- Local and emergency resources, available offline via PWA caching.
- Panic Button for immediate exit that clears sensitive local data.

## Users
- Individuals seeking quick, private assessment and guidance.
- Advocates and community organizations needing a simple, free tool.

## Non-Goals (MVP)
- Complex ML/NLP models.
- Server-side storage of user narratives.
- Geolocation or sensitive background services.

## Roadmap
- Regional resource packs and language localization.
- Expanded keyword sets with expert input.
- Optional encrypted local journal.
